# Helper
Package que j'ai créé pour m'aider lors de mes créations d'applications mobiles avec Flutter. (J'utilise certains packages présent dans pub.dev).

À travers mes nombreuses applications/programmes créé avec le framework Flutter, j'ai constaté que je devais répéter des fonctions. C'est donc décidé de créer un package pour m'aider à développer des applications plus rapidement.

Ce package contient deux dossiers :

• helper : regroupe des classes visant à aider le développeur.  
• helper_extension : regroupe des extensions (String, Map, int...) visant encore une fois à aider le développeur.
